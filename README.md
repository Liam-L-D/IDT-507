# IDT 507 Final Project  
## Information Cultures and Communication Technologies

This project explores the evolution of information cultures and communication technologies, focusing on four major stages: Oral, Print, Electronic, and Digital culture. Each stage is analyzed using key communication theories to understand how information is created, transmitted, and used across time.

## 📌 Project Overview

This project examines how communication technologies shape human behavior, knowledge systems, and social practices. The four major stages of information culture are:

- Oral Culture  
- Print Culture  
- Electronic Culture  
- Digital Culture  

Each section includes:
- Definition  
- Theory Focus  
- Example  
- Personal Analysis  

The project applies the following theories:
- Affordance Theory  
- Diffusion of Innovations  
- Practice Theory  

---

## 🎯 Project Goals

- Understand the evolution of communication systems  
- Analyze how technologies influence information exchange  
- Apply theoretical frameworks to real-world communication systems  
- Present information through a structured web-based format

## 💡 Design Approach

The website is designed with a clean and structured layout to enhance readability and usability. Key design features include:

- Responsive layout using Bootstrap

## 📚 Key Insights

- Communication technologies continuously transform how information is shared and preserved  
- Each stage of information culture has unique affordances and limitations  
- User behavior plays a critical role in shaping how technologies are used  
- Digital culture enables interactive and global communication  

---

## 🚀 Future Considerations

Future developments in communication may involve deeper integration between humans and artificial intelligence, further transforming how information is generated, interpreted, and distributed.
